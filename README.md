# ⚡ AI<span style="color: #2563eb">Fusion</span>: The Neural Synthesis Engine

**AIFusion** is a high-performance, unified AI platform that commands the collective intelligence of the world's most powerful language models—OpenAI, Google Gemini, Meta Llama, and DeepSeek—from a single, premium command center.

Designed for researchers, developers, and power users, AIFusion doesn't just compare models; it synthesizes their outputs, verifies their claims, and optimizes your neural workflow.

---

## 🚀 Core Functionalities

### 1. **Unified Multi-Model Canvas**
Type one prompt and watch it fan out across multiple high-tier AIs simultaneously. 
*   **Parallel Execution**: Real-time response streaming from GPT-4, Gemini 1.5, Llama 3.1, and DeepSeek.
*   **Performance Metrics**: Track latency (seconds) and token efficiency for every model side-by-side.
*   **Neural Consensus**: Our system automatically analyzes all responses to find common ground and highlight discrepancies, effectively filtering out AI hallucinations.

### 2. **Intelligence Pipeline (Workflows)**
Automate complex AI tasks with pre-built neural pipelines.
*   **Code Auditor**: Security-first code reviews.
*   **Creative Architect**: Deep narrative generation.
*   **Legal Analyst**: Risk assessment for documents.
*   **Strategic Planner**: Long-term business modeling.

### 3. **Personalized Knowledge Base (RAG)**
Upload your own documents (PDF, TXT, MD) to create a private knowledge vault.
*   **Neural Grounding**: The AI will prioritize your uploaded data to provide context-aware, hyper-personalized responses.
*   **Private Memory**: Secure storage for your business or research data.

### 4. **AI Battle & Debate Mode**
Force different models to interact with one another.
*   **Refinement**: Models debate a topic until they reach the most logically sound conclusion.
*   **Voting System**: Crown a "Winner" for each session to help train your personal preference model.

### 5. **Premium Mobile Experience**
A fully responsive, mobile-first design featuring:
*   **Neural Drawer**: Quick access to your conversation history (Memory Bank).
*   **Bottom Navigation**: Seamless switching between Chat, Knowledge, Pipelines, and Analytics.
*   **Voice Integration**: Command the core with your voice for hands-free synthesis.

## 🛠️ Technical Architecture

### **Frontend**
*   **Next.js 16 (App Router)**: Lightning-fast SSR and client-side navigation.
*   **TypeScript**: Type-safe development for complex neural state management.
*   **Tailwind CSS**: Custom design system with advanced animations and responsive layouts.
*   **Lucide React**: Premium icon set for intuitive navigation.
*   **NextAuth.js**: Secure authentication with Google, GitHub, and Credentials support.

### **Backend**
*   **Prisma ORM**: Robust database management with MongoDB.
*   **Vercel Edge Rewrites**: High-speed API routing to geographically distributed backends.
*   **Streaming API**: Low-latency communication for real-time AI responses.

## 📦 Getting Started

### 1. Prerequisites
*   Node.js v18+
*   MongoDB Instance
*   API Keys (OpenAI, Google Gemini, etc.)

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/CodeVishal-17/fusion_ai.git

# Install Frontend dependencies
cd frontend
npm install

# Setup Environment
cp .env.example .env
# Fill in your AUTH_SECRET, GOOGLE_ID, GITHUB_ID, etc.

# Start Development
npm run dev
```

### 3. Deployment
The platform is optimized for **Vercel** (Frontend) and **Render** (Backend). Ensure your `vercel.json` rewrites are pointing to your active backend gateway.
